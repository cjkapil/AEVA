Automated EVAluater

This system is an extension of digitization of answer sheets. In this system, along with digitization of answer sheets, even valuation is done by the system itself. The architecture of our project has three main components, OCR, Text Analyzer and Chatbot. All three components have its own significance and purpose to include in this project. This project is the complete design of shifting manual evaluation process to its digitalized form. The first technology of this project is OCR, which is used to scan the handwritten text to its equivalent digitalized text document. Manual Data Collection from a student's exam sheets is always a tedious job which exacts ample amount of time and effort. This image is processed by proposed robust extraction and noise removal algorithm adaptive to environmental conditions. It is then passed through Optical Character Recognition (OCR) system which extracts characters using correlation. Accuracy of system depends on the sample space size of OCR system.After taking digitalized data from the manual answer sheet, which is done by OCR, evaluating subjective answer is a critical task to perform. Since subjective answers have various parameters upon which they can be evaluated such as content, writing style, length of the answer, grammar and keywords. This will be done by Text analyzer.
